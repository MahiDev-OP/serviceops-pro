# ServiceOps Pro

ServiceOps Pro is a portfolio-ready Next.js project built to showcase a strong 2+ years developer profile. It includes:

- A polished marketing homepage
- An API-backed demo request flow
- A responsive operations dashboard
- Interactive work-order stage management with local persistence

## Stack

- Next.js 16 App Router
- TypeScript
- Tailwind CSS 4
- Route Handlers for backend-like form handling

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Pages

- `/` landing page
- `/dashboard` operations dashboard
- `/api/demo` POST endpoint for lead capture
